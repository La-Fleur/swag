			  LCD-style Fonts for the BGI



	 I release these fonts as freeware.

	 These	are  the  first  and the last fonts  I	wrote  using  the
      Borland's  font  editor: I realized that this way of  creating  BGI
      fonts  takes  too much time and patience.  For this reason  I  have
      written  "Easy  Fonts v1.0 for Windows", a program  that	instantly
      converts	any TrueType font to the CHR format used by the BGI.  You
      can find it on my home page.
