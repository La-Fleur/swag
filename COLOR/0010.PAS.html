{
 I have seen a lot of applications that use highintensity
 background colors in Text mode.  How do they do it??????

if you are using an EGA/VGA adapter then you can try :-
}

Procedure SelectIntensity(Intense:Boolean);
Var
  R : Registers;

begin
  if Intense then
    R.BL := 0
  else
    R.BL := 1;
  R.AX := $1003;
  Intr($10, R);
end;
